Magnet Dudes

  • https://jyeung05.itch.io/magnet-dudes
  • Github Repository: https://github.com/Jyeung05/Magnet-Dudes

Magnet Dudes is an indie game created by me and two others. It was programmed in Godot using GDscript. It is a 2D puzzle platformer where you play as 2 dudes and utilize magnets and dimension switching to make it to the exit.




Quest Log

  • Github Repository: https://github.com/Clement-Quach/CalTechHackathon2024

QuestLog is an application created by me and one other. It serves as an app the help people improve their everyday life. It was made using Java. It is an app which provides useful tasks, such as chores, which the user could finish to aquire virtual treats to feed a virtual pet. The virtual pet acts as a motivation for the user to finish the tasks.
